Dictionary (LSJ)

οἴαξ, ᾱκος, Ion. οἴηξ, ηκος, ὁ, prop. A. handle of rudder, tiller (Poll. 1.89), πηδαλίων οἴακος ἀφέμενος (cf. πηδάλιον) Pl. Plt. 272e : generally, helm, οἴακος εὐθυντῆρος ὑστάτου νεώς A. Supp. 717 ; στρέφειν οἴακα E. Hel. 1591 : pl., οἰάκων φύλαξ ib.1578 ; οἴακες εὐπρύμνου νεώς Id. IT 1357 ; τὸν οἴακα εἴσω ἄγειν ἢ ἔξω Pl. Alc. 1.117c. 2. metaph., helm of government, ἐν πρύμνῃ πόλεως οἴ. νωμῶν A. Th. 3 ; πραπίδων οἴ. νέμων Id. Ag. 802 (anap.) ; χαλινῶν ἔργον οἰάκων θʼ ἅμα S. Fr. 869, cf. E. Or. 795 ; τὸν οἴ. στρέφει δαίμων ἑκάστῳ Anaxandr. 4 ; τύχης οἴ. IG 7.3226.5 (Orchom. Boeot.) ; = ὡροσκόπος, Paul.Al. L. 2. II. in Il. 24.269, οἴηκες are prob. rings of the yoke, through which pass the reins for guiding the mules, cf. ἕστωρ.
