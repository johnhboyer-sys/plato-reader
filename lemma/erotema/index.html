Dictionary (LSJ)

ἐρώτημα, ατος, τό, A. that which is asked, question, Th. 3.54, etc. ; ἡ πρὸς τὸ ἐ. ἀπόκρισις ib.60 ; τὰ ἐ. τοῦ ξυνθήματος asking for the password, Id. 7.44 ; ἐ. περί τινος Pl. Prt. 336d ; ἐ. ἐρέσθαι, Id. Phlb. 42e ; διπλᾶ ἔστρεφε τὰ -ήματα Id. Euthd. 276d. 2. in Stoic terminology, a question requiring the answer ‘Yes’ or ‘No’, opp. πύσμα, Chrysipp.Stoic. 2.61. II. in Dialectic, question inviting an answer which may help to refute an opponent, Arist. APr. 64a36(pl.), AP 0.77a36, al.
