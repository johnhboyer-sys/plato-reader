ἐπιτίθημι, Pass. mostly furnished by ἐπίκειμαι : A. Act., lay, put or place upon, of offerings laid on the altar, ἐπὶ μηρία θέντες Ἀπόλλωνι Od. 21.267, cf. 3.179 ; λιβανωτόν Ar. Nu. 426, V. 96, Antipho 1.18 ; set meats on the table, εἴδατα πόλλʼ ἐπιθεῖσα Od. 1.140, cf. 10.355 ; πάντʼ ἐπιθεῖτε on the car, Il. 24.264 ; [νέκυας] ἐπὶ νηυσὶ τιθέντες Od. 24.419 ; τινὶ κύρτον καὶ κώπαν, as a grave-monument, AP 7.505 (=Sapph. 120) : Constr. mostly ἐ. τινί τι, τῷ ἰσχυροτέρῳ πλέον βάρος X. Oec. 17.9, etc. : but also c.gen., ἐ. λεχέων τινά Il. 24.589 ; ἐ. τι ἐπί τινος Hdt. 2.121.δʹ; κεφαλὴν ἐπὶ στέρνα τινός X. Cyr. 7.3.14: c. acc. only, put upon, set up, ἐ. φάρμακα apply salves, Il. 4.190 ; δέελον δʼ ἐπὶ σῆμά τʼ ἔθηκε 10.466 ; στήλην λίθου Hdt. 7.183 ; φάκελον ξύλων E. Cyc. 243 ; ἐ. μνημεῖά τινι to him, Id. IT 702, cf. IG 14.446 (Tauromenium), 12.1068. 2. set upon, turn towards, Ἑκτορέοις ἐπὶ φρένα θῆχʼ ἱεροῖσιν Il. 10.46 ; but τῇ δʼ ἄρʼ ἐπὶ φρεσὶ θῆκε c. inf., put it into her mind to.., Od. 21.1. II. put on a covering or lid, ὡς εἴ τε φαρέτρῃ πῶμʼ ἐπιθείη 9.314 ; κεφαλῇ ἐπέθηκε (as v.l. for ἐφύπερθε) καλύπτρην 5.232 ; λίθον δʼ ἐπέθηκε θύρῃσι, i.e. put a stone as a door to the cave, put it before the door, 13.370 ; also, put a door to, κολλητὰς ἐπέθηκα θύρας 23.194 ; θύρας ἐπέθηκε φαεινάς 21.45 ; θυρεὸν μέγαν 9.240 (v. infr. B. II). 2. set a seal on, BGU 361 iii 22 (ii A.D.) ; apply a pessary, Hp. Steril. 214 (Pass.) ; a cupping instrument, Sor. 2.11 (Pass.). III. put to, add, grant or give besides, ὅσσα τε νῦν ὔμμʼ ἐστὶ καὶ εἴ ποθεν ἄλλʼ ἐπιθεῖτε Od. 22.62, cf. Il. 7.364, etc. ; κράτος, κῦδός τινι, 1.509 (tm.), 23.400 (tm.), 406 (tm.) ; ἡμιτάλαντον χρυσοῦ ib.796. 2. of Time, add, bring on, ἕβδομον ἦμαρ ἐπὶ Ζεὺς θῆκε Od. 12.399 ; μάλα πολλὰ [ἔτεα] Hes. Op. 697.
