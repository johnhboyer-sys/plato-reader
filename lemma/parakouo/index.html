παρακούω, A. hear beside, esp. hear accidentally, hear talk of, Δημοκήδεος τὴν τέχνην Hdt. 3.129 ; ἀξίων λόγου πραγμάτων Pl. Ep. 339e ; παρακήκοα νῦν ὅτι τίκτει AP 5.74 (Rufin.). II. eavesdrop, overhear from, δεσποτῶν ἅττʼ ἂν λαλῶσι Ar. Ra. 750 ; τι παρά τινος Pl. Euthd. 300d ; π. τινός overhear him, Luc. Merc.Cond. 37 ; π. τὸν λόγον Ev.Marc. 5.36.
